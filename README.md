# Clean Architecture Flutter E-Commerce App

## Features

User View

- Products view
- Search Products
- Product AddTo Cart
- Product Purchase
- Add Favorite
- Product Rating
- View Purchase History
- Download Purchase Product
- View Notifications

Admin View

- Admin Dashboard
- Product Add/Edit/Delete
- Category Add/Delete
- User Manage
- Notifications Manage
